## Prerequisites
Install Command Line Tools (~4GB):
```
xcode-select --install
```

## Installation
Run the installation script:
``` 
bash <(curl -L https://raw.github.com/MauricioBalderrama/macOS-Bootstrap/master/install.sh)
```
